**代码块**

+++

在一行需要引用代码,只需要用反引号''引起来就好了

`hello` world                                                        

如果是在一个块内需要引用代码，则在需要引用的代码块的前一行和后一行使用3个反引号，```,同时在前一个反引号后写入代码的语言

```c
#include<stdio.h>
int main()
{
    printf("Hello World!\n");
    return 0;
}
```
