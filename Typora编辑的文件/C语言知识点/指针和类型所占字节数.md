## 指针

---

32 位系统：
指针占用 4 字节（32 位），因为 32 位地址总线最多可寻址 2^32 字节的内存空间。
64 位系统：
指针占用 8 字节（64 位），64 位地址总线理论上可寻址 2^64字节的内存空间。

## 类型

---

> char 				1
> int 					4
> short				2
> long				4 或  8
> long long		8
> float				4
> double			8       



### char

通常占 1 个字节（8 位）。如果是有符号的 char，取值范围是 -128 到 127；
如果是无符号的 unsigned char，取值范围是 0 到 255。
char 主要用于表示字符，也可用于存储小整数。

### int

：通常占 4 个字节（32 位）。有符号的 int 取值范围是 -2147483648 到 2147483647；
无符号的 unsigned int 取值范围是 0 到 4294967295。
int 是最常用的整数类型，用于一般的整数运算。

### unsigned long long：

通常占 8 个字节（64 位），
取值范围是 0 到 18446744073709551615。它是无符号的长整型，
用于需要存储非常大的非负整数的场景，能表示的数值范围比有符号
的长整型更大（因为不需要用一位来表示符号）。
主要区别在于字节长度（表示范围）和是否有符号：
字节长度决定了能表示的数值范围大小，char 最小，unsigned long long 最大。

####  unsigned long long 明确是无符号的，只能表示非负整数，且范围更大。
