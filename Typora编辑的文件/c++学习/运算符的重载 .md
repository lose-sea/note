# 运算的重载

---

```c
struct Stu {
	string name;
	int age;
	int score;
};
int main() {

	Stu s1;
	s1.name = "张三";
	s1.age = 18;
	s1.score = 100;
	cout << s1 << endl; // 错误，无法直接输出结构体变量
	return 0;
}

```

在C++中要对自定义类型的变量,向使用cout 和 << 来输出变量的内容,需要对 << 这个输出运算符进行重载,例如

```c
struct Stu {
	string name;
	int age;
	int score;
};
// 运算符重载
// 重载的是<<(输出运算符),让<<支持Stu类型数据的打印
ostream& operator<<(ostream& os, const struct Stu& s) {
	os << "姓名: " << s.name << endl;
	os << "年龄: " << s.age << endl;
	os << "分数: " << s.score << endl;
	return os;
}
int main() {

	Stu s1;
	s1.name = "张三";
	s1.age = 18;
	s1.score = 100;
	cout << s1 << endl; // 正确，输出结构体变量
	return 0;
}

```
