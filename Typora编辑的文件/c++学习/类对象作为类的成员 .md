# 类对象作为类的成员

C++类的成员可以是另一个类的对象,我们称该成员为对象成员

例如

```cpp
class A {}; 
class B {
    A a; 
}; 
```

B中有对象A作为成员,A为对象成员

```cpp
class Phone {
public :  
    Phone(string name) {
        m_name = name; 
    }
    string m_name;  // 手机品牌名称
}; 

class Person {
public :     
    // Phone m_phone = pName  // 隐式转换法
    Person(string s, string p) : m_name(s), m_phone(p) {  

    }
    string m_name; 
    Phone m_phone; 
}; 
void test() {
    Person p("zhangsan", "apple");  
    cout << p.m_name << " "<< p.m_phone.m_name << endl;
}
```

当其他类对象作为本类成员,构造的时候**先构造类对象**,**再构造自身,**

析构的顺序与构造相反,先析构自身,再析构类对象
