# 空指针访问成员函数

```cpp
class Person {
public: 
    void func() {
        cout << "this person class" << endl; 
    } 
    void showAge() {
        cout << "age = " << age << endl; 
    } 
    int age = 31; 
}; 
void test01() {
    Person* p = NULL; 
    // p->func(); // 正常输出
    p->showAge(); //  报错
}
```

在对象的属性前都会默认加this, 例如

cout << "age = " << age << endl;  实际为   this→age; 

传入的指针为空,就发生了报错

```cpp
    void showAge() { 
        if (this == NULL) {
            return; 
        }
        cout << "age = " << age << endl; 
    } 
```
