# 初始化列表

C++中提供了初始化列表的语法, 用来初始化属性

语法

```cpp
构造函数() : 属性1(值1), 属性2(值2)...{}
```

普通初始化

```cpp
class Person {
public : 
    int m_a; 
    int m_b; 
    int m_c; 
    Person(int a, int b, int c) {
        m_a = a; 
        m_b = b; 
        m_c = c; 
    } 
};  
```

用初始化列表

```cpp
class Person {
public : 
    int m_a; 
    int m_b; 
    int m_c; 
    Person() : m_a(10), m_b(20), m_c(30) {

    }
};  
void test() {
    Person p; 
    cout << p.m_a << " " << p.m_b << " " << p.m_c << endl;  
}
```

让初始化更加灵活,加入参数

```cpp
class Person {
public : 
    int m_a; 
    int m_b; 
    int m_c; 
    Person(int a, int b, int c) : m_a(a), m_b(b), m_c(c) {

    }
};  
void test() {
    Person p(30, 20, 10); 
    cout << p.m_a << " " << p.m_b << " " << p.m_c << endl;  
}
```
