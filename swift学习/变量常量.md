# 变量, 常量

## 变量 var

**定义的变量可以修改** 

```objc
var a:Int = 4
a = 6
print(a)
```

### 变量的定义

```objc
var a: String = "hello world"  

var b: Int = 4

var c: Float = 2.5

var d: Double = 5.6 

var e: Character = "A" 

var f: Bool = true 

var g: Character = "你" 

var h: Character = '你'	// 报错
```

String 还是 Character 都需要使用双引号

**在定义变量的时候,可以不指明变量的类型, 编译器会根据值进行推断**

```objc
var a = 100

print(a)
```

但是已经确定的变量在修改的时候就不可以改变变量的类型了

```objc
var a = 100 
a = 65
a = "hello"	// 报错, a 已经确认时Int类型
```

## 常量 let

**定义的常量不可以修改**  

常量的定义与变量相似, 只需要把 var 替换为 let 就可以了

```objc
let a: Int = 9  
a = 6	// 报错. 常量不可修改
```



## 求变量的类型

`type(of: 变量名)` 

```
var a = 100
print(type(of: a))  // 输出: Int

var b = "hello" 
print(type(of: b))  // 输出: String
```
