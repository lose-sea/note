# 数据容器

一种可以容纳多份数据的数据类型 (容器), 容纳的每一份数据称之为 1 个元素, 每一个元素都可以是任意类型的数据, 如字符串, 数字, 布尔

# 列表

列表是数据容器中的一类, 是一次性可以存储多个数据(元素)的

定义: 

```python
列表名称 = [元素1, 元素2, ...] 

s = [1, 2, 3, 4]
```

特点: 

1. 可以存储不同类型的元素
2. 元素有序 (根据添加时候的顺序) , 可以重复, 元素可以修改

可以根据索引来访问列表中的元素, 正向索引从0开始, 反向索引从 -1 开始, 如果索引是负数, 说明是反向访问

##  基本操作

```python

s = [1, 2, 3, 4, 5, 6, 7]
print(s[-4])    # 4
print(type(s)) # <class 'list'> 

# 修改
s[5] = 'hello' 
print(s[5]) # hello 

# 打印列表
print(s) # [1, 2, 3, 4, 5, 'hello', 7] 
for i in s : 
    print(i, end = " ")  # 1 2 3 4 5 hello 7 
print()
# 删除  del
del s[3] 
print(s) # [1, 2, 3, 5, 'hello', 7]
```

可以用`print (列表名)` 直接打印整个列表 

## 切片

切片是指对操作的数据截取其中一部分的操作, 列表, 字符串, 元组都支持切片操作

### 语法: 

```python
序列数据[开始索引: 结束索引: 步长]
```

1. **不包含结束索引位置对应的元素** (开始索引未指定默认为 0; 结束索引未指定默认为列表长度(直到列表末尾); 步长未指定默认为 1 ) 
2. 索引采用正向, 反向索引都可以 
3. 步长是选取间隔, 默认步长为 1 

## 列表常见方法

| 方法      | 作用                                                         | 样例               |
| --------- | ------------------------------------------------------------ | ------------------ |
| append()  | 在列表的尾部追加元素                                         | s.append(10086)    |
| insert()  | 在指定索引之前，插入该元素                                   | s.insert(0, 92)    |
| remove()  | 移除列表中第一个匹配到的值                                   | s.remove(75)       |
| pop()     | 删除列表中指定索引位置的元素（如果未指定索引，默认删最后一个） | s.pop(2) / s.pop() |
| sort()    | 对列表进行排序（列表元素的数据类型一致，才可以进行排序）     | s.sort()           |
| reverse() | 反转列表元素                                                 | s.reverse()        |

**sort排序要求所以的元素都是同一类型**

```python
s = [45, 56, 98, 5, 35]  
s.sort()
print(s)  #[5, 35, 45, 56, 98]
```

#### 合并列表

```python
# 合并两个列表, 并对合并后的列表去重
num1 = [12, 56, 87, 99, 87] 
num2 = [12, 99, 556, 36, 85, 87] 

print(num1) 
print(num2) 

for i in num1: 
    num2.append(i) 
print(num2)  # [12, 99, 556, 36, 85, 87, 12, 56, 87, 99, 87]  未去重

num_list = []  # 记录去重后的列表元素
 
for i in num2: 
    if i  not in num_list:  # in 判断元素是否存在于列表中, 存在返回 True, 否则返回 Flase 
        # not 取反
        num_list.append(i) 

print(num_list) # [12, 99, 556, 36, 85, 87, 56]
```

也可以直接用 + 将两个列表合并

```python
num1 = [12, 56, 87, 99, 87] 
num2 = [12, 99, 556, 36, 85, 87] 

num_list = num1 + num2 

print(num_list)
```

## 解包 组包

解包: 将列表这一类容器解开成一个独立的元素 

组包: 将多个值合并到一个容器

```python
num1 = [12, 56, 87, 99, 87] 
num2 = [12, 99, 556, 36, 85, 87] 

num_list = [*num1, *num2]   # 解包

print(num_list)  # [12, 56, 87, 99, 87, 12, 99, 556, 36, 85, 87]
```

### 案例: 生成1 - 20 的平方组成的列表

#### 法一: 传统方法

```python
法一: 
num_list = [] 
for i in range(1, 21): 
    num_list.append(i ** 2) 
print(num_list)
```

#### 法二: 列表推导式

格式: 

```python
[要插入的值 for i in list] 
```

```py
num_list = [] 
num_list = [i ** 2 for i in range(1, 21)]
print(num_list)
```



### 案例: 生成1 - 20 所有偶数平方的列表 

#### 列表推导式: 

```python
num_list = [] 
num_list = [i ** 2 for i in range(1, 21) if i % 2 == 0] 
print(num_list) # [4, 16, 36, 64, 100, 144, 196, 256, 324, 400]
```



# 字符串

字符串是字符的容器, 一个字符串中可以存放任意数量的字符 

特点: **不可变性 (无法修改)**, 有序性, 可迭代性 

字符串中每一个字符元素都有具有其对应的下标 (索引), 通过元素对应的索引, 就可以获取对应的元素 

> 从前向后,(正向索引), 下标从 0 开始, 从后向前 (反向索引), 下标从 -1 开始
