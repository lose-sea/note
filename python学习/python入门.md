## 字面量

### 整型 int		2 ,  5

### 浮点数 float		1.2

### 布尔 bool 				True, Flase (注意首字母大写)

布尔类型本质是数字类型, 在涉及到数学运算的时候, 会自动将 True 转为 1 , 将 Flase 转为 0 

### 字符串 str			“hello world”

### 空值 (None Type)			None

### 数据容器		数组, 元组, 集合, 字典

```python
print(True) # True
print(3.14) # 3.14
print(True + 1) # 2 
print(False)    # False
print(None) # None 
```
