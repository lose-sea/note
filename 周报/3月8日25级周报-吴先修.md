# 本周学习内容

1. 学习了Objective-C的块的概念及基本语法(块的定义, 块变量的创建, 有参块,无参块的定义和调用, 块语局部变量的关系, 使用__block修饰局部变量可以在块内修改, 使用块作为方法的参数 ) 
1. Objective-C中类和对象的基本语法(类的声明和实现, 方法的声明和实现, 对象的产生和使用, 方法的调用(通过对象调用, 通过类调用), 形参可变的方法) 
1. 封装的基本了解 (访问控制符(@public, @package, @protect, @private), 合成setter方法和 getter方法, 系统自动合成setter 和 getter方法, 特殊指示符(copy, nonatomic, assign, readonly, retain))
