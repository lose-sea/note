## 字符串存储位置总结

### 一、常量区

```objc
// 字面量，编译期确定，不管多长都在常量区
NSString *str1 = @"hello";
NSString *str2 = @"hello1234567890"; // 再长也在常量区

// stringWithString/initWithString 传字面量 → 编译器优化，同常量区地址
NSString *str3 = [NSString stringWithString:@"hello"];
NSString *str4 = [[NSString alloc] initWithString:@"hello"];

NSLog(@"%p %p %p %p", str1, str2, str3, str4);
```

------

### 二、Tagged Pointer（指针中）

```objc
// stringWithFormat/initWithFormat 且最终字符数 ≤ 9
NSString *str1 = [NSString stringWithFormat:@"hello"];       // 5个字符
NSString *str2 = [NSString stringWithFormat:@"hello1234"];   // 9个字符
NSString *str3 = [[NSString alloc] initWithFormat:@"hi"];    // 2个字符

// 注意：看的是最终生成的字符串长度，不是格式串
NSString *str4 = [NSString stringWithFormat:@"%d", 123];     // 最终"123" 3个字符 → Tagged Pointer
NSString *str5 = [NSString stringWithFormat:@"%@", @"hi"];   // 最终"hi" 2个字符 → Tagged Pointer
```

------

### 三、堆区

```objc
// stringWithFormat/initWithFormat 且最终字符数 > 9
NSString *str1 = [NSString stringWithFormat:@"hello12345"];  // 10个字符
NSString *str2 = [NSString stringWithFormat:@"%@", @"hello12345"]; // 最终10个字符

// initWithUTF8String 任意长度都在堆区
NSString *str3 = [[NSString alloc] initWithUTF8String:"hello"];

// NSMutableString 任何方式都在堆区
NSMutableString *str4 = [NSMutableString stringWithString:@"hello"];
NSMutableString *str5 = [NSMutableString stringWithFormat:@"hi"];
NSMutableString *str6 = [[NSMutableString alloc] init];
```

## NSStringWithString 的存储规则

### 入字面量 → 常量区（编译器优化）

```objc
NSString *str = [NSString stringWithString:@"hello"];
NSLog(@"%@", [str class]); // __NSCFConstantString
NSLog(@"%p", str);         // 和 @"hello" 同一地址，常量区
```

### 传入变量 → 看变量本身

```objc
// 传入 Tagged Pointer 字符串
NSString *tmp = [NSString stringWithFormat:@"hi"]; // Tagged Pointer
NSString *str = [NSString stringWithString:tmp];
NSLog(@"%@", [str class]); // NSTaggedPointerString，还是 Tagged Pointer

// 传入堆区字符串
NSString *tmp2 = [NSString stringWithFormat:@"hello12345"]; // 堆区
NSString *str2 = [NSString stringWithString:tmp2];
NSLog(@"%@", [str2 class]); // __NSCFString，堆区
```

### 总结规律
> stringWithString: 传字面量  → 常量区（直接返回同一对象）
> stringWithString: 传变量   → 和传入的变量存储位置一致

**`stringWithString:` 本质是复制传入的字符串，传什么类型就返回什么类型，传字面量编译器直接优化返回同一对象，并不总是在堆区。**
