## 面向过程与面向对象的区别

### 定义

**面向过程（POP - Procedure Oriented Programming）** 以**函数**为基本单位，将问题分解为一系列步骤，按顺序依次执行。程序 = 数据结构 + 算法。 

面向对象的三大特性: 

1. 封装 
2. 继承
3. 多态

**面向对象（OOP - Object Oriented Programming）** 以**对象**为基本单位，将数据和行为封装在一起，通过对象之间的交互来解决问题。程序 = 对象 + 消息。



面向过程是一种**自顶向下**的编程思想，适合逻辑简单、规模较小的程序；面向对象是一种**自底向上**的编程思想，通过封装、继承、多态降低了大型软件的开发和维护成本，是现代软件开发的主流范式。
