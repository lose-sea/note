# 结构体

```objective-c
#import<Foundation/Foundation.h>
int main(int argc, char* argv[]) {
    @autoreleasepool {
        struct rect {
            int x;
            int y;
            int width;
            int height;
        }rect1 = {1, 3, 4, 5};
        //  在定义结构体的同时, 还可以定义结构体变量
        // 可以直接对结构体变量执行初始化
    }
    return 0;
}
```

> 在定义结构体的同时, 还可以定义结构体变量 
>
> 可以直接对结构体变量执行初始化
